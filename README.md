# Polanco Inversiones · Flipping

Aplicación web instalable para analizar y gestionar operaciones de flipping inmobiliario.

## Funciones

- Precio máximo de compra, utilidad, margen, ROI y decisión PASA / NO PASA.
- Sensibilidad de compra y venta.
- Escenarios de financiación.
- Base local de operaciones, fotos, pipeline y comparador.
- Exportación CSV, copia JSON e informe PDF.
- Instalación en iPhone y Android como PWA.

## Datos

Las operaciones se guardan localmente en el navegador mediante IndexedDB. No existe sincronización automática entre dispositivos en esta versión.

## Publicación

El workflow `.github/workflows/pages.yml` publica automáticamente cada actualización mediante GitHub Pages.
